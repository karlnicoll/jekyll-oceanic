This is a sample blog. The posts are all taken from the minima template and should give you an idea of what text looks like. Feel free to take a look around.

The pagination is done using the jekyll-paginate-v2 plugin , which is fully supported by the template. The paginator is only configured for 2 posts per page so that you can see the paginator at the bottom of the page. This can be increased easily by changing the per_page paginator configuration setting.
